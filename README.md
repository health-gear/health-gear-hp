テクノロジーとデータの力で、行政・医療という「社会インフラ」のアップデート（DX）を推進する次世代のシビックテック・カンパニー「**株式会社ヘルスギア（Health Gear, Inc.）**」の公式コーポレートサイトのソースコード・リポジトリです。

本リポジトリは、シビックテック・コミュニティ「[Code for Matsubara](https://codeformatsubara.org)」のポータルサイトと同じ環境でホスティング・管理されており、地域課題を解決するためのオープンな技術実装のハブとして機能しています。

## 📖 Overview

**「政治と医療にcodeを流す - Breathing new life -」**

株式会社ヘルスギアは、地域社会の基盤となる行政システムや医療インフラに存在する「バグ（非効率やアナログな課題）」を、モダンなテクノロジー（生成AI、Webフレームワーク）によって修正・最適化する専門家集団です。

机上の空論ではなく、自らシステムを設計・開発し、現場に「実装」する圧倒的な開発力を強みとしています。

## ⚡ Core Business Domain

当社の事業領域および、本サイトで紹介している主要なソリューションです。

### 1. GovTech＆シビックテック開発
行政の非効率を解消するシステムや、市民参加を促すアプリケーションの企画・開発・実装。
* **代表的プロジェクト**: 
  * 匿名陳情AIシステム「令和ボイス」（世界三大デザイン賞エントリー予定）
  * 松原市議会議事録AI検索システム（FastAPI / Gemini AI を用いた高度検索・要約エンジン）

### 2. データ駆動型パブリックアフェアーズ
データ解析や有権者インサイト分析を駆使した次世代の政策立案・広報支援。デジタルキャンペーンのトータルサポート。

### 3. 地域医療DX（ヘルスケア・イノベーション）
地域医療の拠点となる薬局等のDX支援。業務フローの最適化により、医療従事者が対人業務に集中できる健全なシステム環境の構築。

## 🛠 Tech Stack & Architecture

本コーポレートサイトは、高速なレンダリングと高いメンテナビリティ、そしてセマンティックウェブ（LLMO/SEO）を意識して構築されています。

* **Markup & Style**: HTML5 / Tailwind CSS (CDN)
* **Design System**: StripeライクなクリーンUI（Light Mode）、レスポンシブデザイン
* **Typography**: Inter / Noto Sans JP
* **Semantic Web**: 
  * Schema.org (`Organization`, `SoftwareApplication`, `Person`, `Service`) による構造化データ実装
  * OGP (Open Graph Protocol) への完全対応

## 💡 About LLMO (Large Language Model Optimization)

本コーポレートサイトのソースコード（HTML）は、人間にとっての視認性や美しいUI/UX（Tailwind CSSによる実装）を提供するだけでなく、検索エンジンや生成AI（LLM）が企業情報を正確にクロール・理解できるように、Schema.orgを用いた厳密な構造化マークアップが施されています。
